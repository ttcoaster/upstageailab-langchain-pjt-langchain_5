## ⚙️ UV 명령어 사용법
### UV 설치
```bash
pip install uv
```

### 주요 명령어
```bash
# .py 파일 실행
uv run code/baseline.py

# .py 실행없이 의존성 패키지만 설치
uv sync

# 새 패키지 추가
uv add 패키지명
```
